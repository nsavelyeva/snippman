How to make SnippMan to interact with Google SpreadSheets

Please note, work with Google SpreadSheets is possible if:

  • You have created a Google project account -
    can be created at https://console.developers.google.com/cloud-resource-manager.
  • You have Google Drive API enabled - can be done using the above link,
    please also see https://developers.google.com/drive/v3/reference/
  • You have Google SpreadSheets API enabled - can be done using the very first link,
    please also see https://developers.google.com/sheets/api/reference/rest/
  • You have properly configured SnippMan application here as follows:
    • uploaded the private key file provided by Google - it will be kept in snippman/conf folder as gdocs.json.
    • set your e-mail address in the EMAIL property - it will be kept in snippman/conf/custom.json file.

    It is not recommended to change settings manually in snippman/conf/* files, please use this link instead.
